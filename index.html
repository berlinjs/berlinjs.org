Berlin.JS is a usergroup focused on JavaScript and related topics. We meet regularly on the 3rd Thursday each month at 7pm at co.up Offices, Adalbertstraße 7-8 in Berlin-Kreuzberg. Talks start at 7:30pm.

Next meetup on August 15 th

Embedded native code inside the web browser — C++ goes Web

Martina Kraus
@MartinaKraus11

Porting an old C++ desktop app to a single page application without rewriting business logic?

That's possible with web assembly. Further running native embedded Code inside the browser comes without a performance penalty. This enables the developer to create also compute heavy tasks like video and image rendering. After small examples, aiming to show the compiler toolchain, Martina will deep dive into the question: How to port existing C++ Application to web assembly?

DX & styling applications

Anton Kastritskiy
@AntonK52

The talk is about a home grown tool called reshadow. It makes it easy to style modern web application a lot like styled components yet with out the performance compromises. It brings together the developer experience of modern CSS-in-JS solutions and the performance of CSS modules.

Composing better predicates in JavaScript

Rudi Yardley
@rudiyardley

We often have to tradeoff clarity, precision and brevity when dealing with predicate expressions in JavaScript. In this lightening talk Rudi will quickly present an easier way to think about and manage predicates.
