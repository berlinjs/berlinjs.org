Berlin.JS is a usergroup focused on Javascript and related topics. We meet regularly on the 3rd Thursday each month at 7p.m. at co.up Offices, Adalbertstraße 7-8 in Berlin-Kreuzberg.

Next meetup on January 17th

Acorn: A ridiculously fast JavaScript parser

Marijn Haverbeke (@marijnjh)

A short foray into the anatomy of a recursive-descent parser, literate programming, obsessive JavaScript optimization, and open-source machismo.

Vert.x and Encrypted Profile Documents

Philipp Brüll (@bphifty)

Introducing a webapp that is build on Vert.x and Encrypted Profile Documents. The technology ships around some issues with privacy, security and load balancing.

Re-writing a large JS app to AMD/require.js

Karl Westin (@karlwestin)

How do you go from static file loading to modules and AMD? A short talk about how to start, plenty of weird bugs, testing and substack philosophy.

PouchDB

Jan Lehnardt (@janl)

A cross-browser, offline-enabling, mobile-ready, pure-javascript implementation of CouchDB — can it be true?

It sure can! PouchDB addresses the pain of data synchronisation in loosely connected networks (Hello Internet, especially mobile!).

This talk tells you all you need to know to get started buidling compelling user experiences with data being available anywhere.
