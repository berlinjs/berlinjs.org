How to implement WebSockets in nodejs

Bodo Kaiser

Node.js is a great environment which offers good tools for handling I/O. Unfortunately there are a lot of smaller not well supported (web-)protocols. I want to explain how easy it is to implement an "unsupported" protocol.

As example I want to present my node-websockets implementation (I know it is actually a bad example for an "unsupported" protocol) and share the experience I made during this journey.
