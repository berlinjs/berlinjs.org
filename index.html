Berlin.JS is a usergroup focused on Javascript and related topics. We meet regularly on the 3rd Thursday each month at 7p.m. at co.up Offices, Adalbertstraße 7-8 in Berlin-Kreuzberg.

Next meetup on March 21st

Audio Live Coding

The Web Audio API is a fascinating piece of technology that enables realtime audio output in Browsers. Together with a few other pieces of technology Jan Krutisch built a simple environment to do live coding in the context of a live music performance.

I'll perform four songs for you:

  1. The Web Audio API
  2. Music was my first love
  3. A story of liv3 c0ding
  4. Impro-vision

Step by step guide to simple meteor.js app

Meteor is an open-source platform for building top-quality web apps in a fraction of the time.

Kris Urbas will show you how easy it is to start developing with meteor by live coding simple web chat application.

Check this short screencast to get excited!
